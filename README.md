## Got
https://github.com/sindresorhus/got

```js
import got from 'got';

try {
	const response = await got('https://sindresorhus.com');
	console.log(response.body);
	//=> '<!doctype html> ...'
} catch (error) {
	console.log(error.response.body);
	//=> 'Internal server error ...'
}
```
